# KIWI-GO
Офицальный GIT репозиторий игры KIWI GO. 
# Правила для разрабов:
1) Если вы хотите внести именения в игру ,то всегда делайте отчет в лог
2) НЕ трогайте папки и файлы репозитория если вы не админ, за исключением игры и её ресурсов
3) Пожалуйста не показывайте игру и все её ресурсы без ведома админа друзьям и людям не относящиеся к разработке
